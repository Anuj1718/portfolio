# Portfolio

A personal portfolio website built with modern web technologies.

## About

This is my portfolio website where I showcase my projects, skills, and experience.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn-ui

## Getting Started

Clone the repository and install dependencies:

```sh
git clone https://github.com/Anuj1718/portfolio.git
cd portfolio
npm install
```

Run the development server:

```sh
npm run dev
```

Build for production:

```sh
npm run build
```
